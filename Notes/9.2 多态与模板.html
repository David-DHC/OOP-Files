  2. 动多态和静多态：

    1. 动多态：基于继承和虚函数的多态处理。由于涉及晚绑定，因此在运行期处理。

    2. 静多态：基于模板设计的多态处理。在编译期处理。

  4. 函数模板和类模板

    1. 函数模板：某些函数的实现与形参类型关系不大，此时可将所有这些函数统一写为一个泛型函数，即函数模板。

      示例：

      xxxxxxxxxx
      7

      template <typename T> // 定义下面sum函数为不定类型的泛型函数，typename也可写作class
      2
      T sum(T a, T b) 
      3
      { 
      4
          return a + b; 
      5
      }
      6
      ​
      7
      cout << sum(1.2, 1.4) << " " << sum(1, 1) << " " << sum<int> (1,1.2) << endl;

      注释1：函数语句必须满足函数的要求。

      注释1示例：上述 sum(a, b); 的实现中用到了'+'这个符号，那么调用sum函数的T类型中应该有'+'的定义。

      注释2：函数可以指定传参的类型。

      注释2示例：见上sum<int>(1,1.2)一句。
